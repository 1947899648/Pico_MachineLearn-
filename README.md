## 🚀 快速开始 / Quick Start

1. 前往 [Releases](https://github.com/1947899648/Pico_MachineLearn-/releases) 下载最新 APK 包
2. 通过 `adb` 或 Pico 开发者工具安装到设备
3. 戴上 Pico VR 头显运行体验

```bash
# 安装到 Pico 设备
adb install Pico_MachineLearn.apk
```

> 如需要源码工程，请检查仓库根目录备份文件或联系作者。
